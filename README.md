# Pixel War .io

Pixel War is an io game. It is running at https://pixelwar.nethacker.cloud .
Players have a rifle first, and they can join a server, or create a secret server.

Here's some plugin: https://github.com/NeuralNexusLab-nh/PixelWar/tree/main/plugin
To use plugin, you need input them to devtool's console.
